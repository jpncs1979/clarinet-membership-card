# 🚀 クイックスタートガイド

## 3ステップで起動！

### ステップ1: パッケージをインストール
PowerShellで以下を実行：
```powershell
npm install
```

### ステップ2: サーバーを起動
```powershell
npm start
```

### ステップ3: ブラウザで開く
ブラウザで以下にアクセス：
```
http://localhost:3000
```

## 🎯 使い方

1. **ログイン**: メールアドレスと誕生日を入力して「ログイン」ボタンをクリック
2. **会員証表示**: ログイン成功後、会員証が表示されます

## 📂 シクミネットCSVで会員を更新する

1. シクミネットから会員一覧CSVをダウンロードする
2. `data/sikuminet.csv` に保存する
3. 次のコマンドを実行する：
   ```powershell
   node scripts/import-sikuminet-csv.js
   ```
4. サーバーを再起動する（Ctrl+C → `npm start`）

詳しくは **`docs/CSVの使い方.md`** を参照してください。

---

## ⚠️ トラブルシューティング

### npm installが失敗する場合
管理者権限でPowerShellを開いてから実行してください。

### ポート3000が使用中の場合
`.env`ファイルの`PORT=3000`を`PORT=3001`に変更し、`http://localhost:3001`にアクセスしてください。
